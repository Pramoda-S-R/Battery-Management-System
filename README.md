# Battery Management System using BQ76925
The repository includes - Circuit diagrams, Code for the hardware used (Arduino and ESP8266 module), Features and Specifications.

## Requirements
1. BQ76925 IC
2. Arduino UNO R3
3. NODEMCU ESP8266
4. Blynk Account
5. 3 - Li-Ion(3.7V,200 mAh) Batteries
6. 5V Power Supply
7. Resistors
     - 100Ω   - 5
     - 1kΩ    - 3
     - 4.7kΩ  - 2
8. Capacitors
     - 1µF    - 6
     - 1000pF - 2
     - 10µF   - 1
     - 0.1µF  - 1
